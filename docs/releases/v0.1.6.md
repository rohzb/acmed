# acmed v0.1.6 Release Notes

Release date: 2026-04-19

## Summary

This patch release focuses on repository security checks and release
automation.

## Highlights

- Added continuous secret scanning in CI using TruffleHog and Gitleaks.
- Added release workflow on `vX.Y.Z` tags to:
  - validate tag/version/changelog consistency
  - run tests
  - build Python artifacts
  - build and push Docker images to GHCR
  - create a GitHub Release with attached artifacts
- Hardened scanner reproducibility by pinning TruffleHog and Gitleaks scanner
  references.

## Upgrade Notes

- No runtime API or configuration contract changes.
- Repository maintainers can cut a release by pushing an annotated SemVer tag
  that matches `pyproject.toml`.

## Validation

- Unit test suite passes for this release candidate.
- Release workflow verifies tag/version/changelog coherence before publishing.
