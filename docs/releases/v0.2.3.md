# acmed v0.2.3 Release Notes

Release date: 2026-04-26

## Highlights

- Added `--force` to `acme.sh` issue invocation in the built-in `acme_sh`
  issuer backend.

## Operational Impact

- Repeated issuance requests for an already-issued domain now continue through
  renewal flow instead of returning `issuer failed with exit code 2`.
