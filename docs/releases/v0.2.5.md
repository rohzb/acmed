# acmed v0.2.5 Release Notes

Release date: 2026-04-26

## Highlights

- Added `force_renew` issuer option for `acme_sh`.
- Default path reuses existing cert state without forced renewal.

## Configuration

Example issuer snippet:

```yaml
issuers:
  - name: le-hetzner-dns
    type: acme_sh
    executable: /usr/local/bin/acme.sh
    challenge_mode: dns-01
    plugin_name: dns_hetznercloud
    force_renew: false
```

Set `force_renew: true` only when you explicitly need forced reissuance.
