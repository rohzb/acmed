# acmed v0.2.6 Release Notes

Release date: 2026-04-26

## Highlights

- Moved Docker issuer enablement to an addon-driven runtime model.
- Added plugin addon installer script for deterministic dependency setup.
- Simplified compose issuer presets around `ACMED_PLUGIN_DIRS`.

## Operational Impact

- Docker builds now use a single runtime image target with optional addon
  installation, instead of maintaining separate runtime targets per issuer CLI.
- Issuer-enabled local runs and Pebble test profiles select addons explicitly
  with `ACMED_PLUGIN_DIRS` and inherit the acme.sh default ref `3.1.2`.
